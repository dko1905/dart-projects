A library for Dart developers.

## Usage

A simple usage example:

```dart
import 'package:dartToCharCode/dartToCharCode.dart' as ng;

main() {
	print(ng.stringToIntList('a b c d e f g'));
}
```
It returns:
```sh
	[97, 32, 98, 32, 99, 32, 100, 32, 101, 32, 102, 32, 103]
```
